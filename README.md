# 💧 Last Drop: Gamified Irrigation Optimization

**2nd Place Winner at Gamification of Optimization Hackathon (among 25 teams)**

## 📌 Project Overview
"Last Drop" is an interactive, gamified optimization project designed to make water-efficient irrigation decisions. This project demonstrates how complex mathematical modeling can be integrated with user-centric gamification to solve real-world sustainability challenges. 

*Featured on the official GAMS blog for applying operations research to an interactive problem.*

## 🛠️ Tech Stack & Methods
* **Optimization Engine:** GAMS, CPLEX
* **Modeling Technique:** Mixed-Integer Programming (MIP) / Mathematical Optimization
* **Core Focus:** Resource allocation, gamified decision-making comparison

## 🚀 What it Does
1. **Optimal Baseline:** Solves the irrigation water allocation problem under constraints to find the mathematically optimal water usage.
2. **User Interaction:** Allows users to input their own irrigation strategies within a gamified interface.
3. **Comparative Analytics:** Compares the user's manual strategy against the GAMS/CPLEX optimal solution, highlighting inefficiencies and teaching optimal resource management.

## 💼 Business Value
While focused on irrigation, the underlying mathematical architecture of this project is directly applicable to **supply chain routing, inventory allocation, and production planning**—proving that optimization can be both rigorous and accessible.
